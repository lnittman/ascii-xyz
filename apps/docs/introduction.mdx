## platform components

arbor consists of several integrated components:

- **web application**: main interface for chat, projects, and settings
- **desktop daemon**: optional local integration for file system access
- **ai service**: backend service powering intelligent responses
- **mobile app**: ios application for on-the-go access

## who is arbor for?

arbor is designed for:

- **engineers** building ai-powered applications
- **teams** collaborating on ai projects
- **developers** integrating ai into their workflows
- **anyone** seeking powerful ai assistance with context persistence

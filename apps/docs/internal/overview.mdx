## repository structure

```
arbor-xyz/
├── apps/
│   ├── app/          # main web application
│   ├── api/          # api service
│   ├── ai/           # ai service (mastra)
│   ├── daemon/       # desktop daemon
│   └── docs/         # this documentation
├── packages/
│   ├── @repo/api/    # domain services
│   ├── @repo/database/  # prisma schema
│   ├── @repo/design/    # ui components
│   ├── @repo/auth/      # authentication
│   └── @repo/mastra/    # ai client
└── docs/            # (deprecated - moved here)
```

## development workflow

### local development
```bash
# install dependencies
pnpm install

# run migrations
pnpm db:migrate

# start all services
pnpm dev
```

### testing
```bash
# run tests
pnpm test

# run specific tests
pnpm test:app
pnpm test:ai
```

### deployment
```bash
# build all apps
pnpm build

# deploy to vercel
vercel deploy

# deploy ai to mastra cloud
pnpm deploy:ai
```

## environment configuration

critical environment variables:
- `DATABASE_URL` - postgresql connection
- `CLERK_SECRET_KEY` - authentication
- `OPENAI_API_KEY` - ai provider
- `NEXT_PUBLIC_AI_URL` - ai service url

see [environment variables](/internal/operations/state/environment-variables) for complete list.
